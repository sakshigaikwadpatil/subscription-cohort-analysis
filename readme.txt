# Customer Subscription Cohort & Revenue Analysis Using SQL

## Project Overview
This project analyzes subscription customer behavior using PostgreSQL.

The analysis focuses on:
- Revenue trends
- Customer churn
- Retention analysis
- Cohort analysis
- Subscription performance

## Tools Used
- PostgreSQL
- pgAdmin
- SQL

## Key Analysis
- Total Revenue KPI
- Monthly Revenue Trends
- Customer Acquisition Analysis
- Churn Rate Analysis
- Cohort Retention Analysis
