# Customer Subscription Cohort & Revenue Analysis Using SQL

## Project Overview
This project analyzes subscription customer behavior using PostgreSQL.

The analysis focuses on:
- Revenue trends
- Customer churn
- Retention analysis
- Cohort analysis
- Subscription performance

## Tools Used
- PostgreSQL
- pgAdmin
- SQL

## Key Analysis
- Total Revenue KPI
- Monthly Revenue Trends
- Customer Acquisition Analysis
- Churn Rate Analysis
- Cohort Retention Analysis

## Skills Demonstrated
- SQL Query Writing
- Data Cleaning
- Aggregations
- CTEs
- Business KPI Reporting
- Cohort Analysis

## Business Problem
The objective of this project is to analyze customer subscription behavior and identify trends related to:
- customer churn
- retention
- customer acquisition
- subscription revenue

## Dataset Information
The dataset contains:
- customer IDs
- subscription start dates
- cancellation dates
- subscription pricing
- payment status
- subscription plans

## Business Insights
- Identified customer churn patterns using cancellation data.
- Analyzed subscription revenue trends across customer cohorts.
- Evaluated retention performance based on signup periods.
- Compared active and churned customer behavior.

## SQL Concepts Used
- Aggregate Functions
- GROUP BY
- CTEs
- Date Functions
- Filtering
- Cohort Analysis
